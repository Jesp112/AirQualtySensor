# AirQualtySensor
An indoor air quality sensor using an arduino/esp32


# Prerequisites 

* Hardware
    Arduino or ESP32 of your choice.
    ENS160 air quality sensor of your choice (Link to the one i use) https://ardustore.dk/produkt/ens160aht21-luftkvalitet-temp-fugt-module

# Installation
    1. Clone repo
    git clone https://github.com/Jesp112/AirQualtySensor.git
    
    2. Download libarys 
        - Adafruit ENS160 – by Adafruit - https://ardustore.dk/error/DFRobot_ENS160-main.zip
        - LiquidCrystal – by ARduino 

    3. Upload AirQuality to arduino/esp32
